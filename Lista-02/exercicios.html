  • Malware Software malicioso, como vírus, worms, ca Troia, ransomware, spyware, etc., que pode causar danos aos sistemas e dados.
  • Firewall Uma barreira de segurança que monitora e controla o tráfego de rede, permitindo ou bloqueando com base em regras predefinidas.

  • Engenharia Social A manipulação psicológica de pessoa para obter informações confidenciais.
  • Política de Segurança Um conjunto de diretrizes e regras para proteger os ativos de uma organização.

Exercício 5

  • Mantenha seu software atualizado Mantenha seu sistema operacional, navegador, plugins e aplicativos de segurança sempre atualizados. As atualizações geralmente corrigem vulnerabilidades de segurança.
  • Use um navegador seguro Navegadores como Chrome, Firefox e Edge são conhecidos por suas medidas de segurança. Certifique-se de que seu navegador esteja atualizado.
  • Instale um antivírus confiável Utilize software antivírus e antimalware de qualidade e mantenha-o atualizado. Isso ajuda a proteger contra ameaças online.
  • Use uma conexão segura Evite usar redes Wi-Fi públicas não seguras para transações sensíveis ou acesso a contas pessoais. Prefira redes VPN (Rede Privada Virtual) ao usar redes públicas.
  • Verifique a autenticidade dos sites Certifique-se de que o site que você está visitando seja legítimo. Verifique se o URL começa com "https://" e procure por um ícone de cadeado na barra de endereço.
  • Cuidado com phishing Esteja atento a e-mails, mensagens e links suspeitos. Não clique em links de fontes não confiáveis e não forneça informações pessoais em resposta a e-mails suspeitos.
  • Use senhas fortes Utilize senhas únicas e fortes para suas contas online. Considere o uso de um gerenciador de senhas para manter suas senhas seguras.
  • Habilite a autenticação em duas etapas (2FA) Ative a autenticação em duas etapas sempre que possível para adicionar uma camada extra de segurança às suas contas online.
  • Evite downloads suspeitos Não faça o download de arquivos ou programas de fontes não confiáveis. Use apenas lojas de aplicativos oficiais para dispositivos móveis.
  • Esteja ciente de cookies e rastreamento Configure suas configurações de privacidade do navegador para controlar como os cookies são armazenados e gerenciados. Use extensões de bloqueio de rastreamento, se desejar.
  • Eduque-se Esteja ciente das ameaças online mais recentes e das táticas de hackers. A educação contínua é fundamental para manter-se seguro na web.
  • Faça backup de seus dados Faça regularmente backups de dados importantes para evitar a perda de informações em caso de ataques cibernéticos.
  • Use uma rede privada virtual (VPN) Uma VPN ajuda a proteger sua privacidade online ocultando seu endereço IP e criptografando sua conexão. É especialmente útil ao usar redes Wi-Fi públicas.

Exercício 6

Phishing Ataque de Negação de Serviço (DDoS) Ransomware Malware Ataque de Engenharia Social Injeção SQL Ataque de Força Bruta
O phishing é um ataque que envolve o uso de e-mails falsificados, sites ou mensagens para enganar as vítimas a revelar informações confidenciais, como senhas e números de cartão de crédito. Os e-mails de phishing muitas vezes se disfarçam de fontes confiáveis para atrair as pessoas. Nesse tipo de ataque, um grande número de dispositivos é usado para inundar um servidor ou rede com tráfego, sobrecarregando-o e tornando-o inacessível para os usuários legítimos. O ransomware é um malware que criptografa os arquivos do sistema e exige um resgate (geralmente em criptomoedas) para descriptografá-los. Os dados da vítima são mantidos como reféns até que o resgate seja pago. Malware é um termo genérico para software malicioso que inclui vírus, cavalos de Troia, spyware e outros programas maliciosos. Eles são projetados para infectar e danificar sistemas ou roubar informações. Os ataques de engenharia social exploram a manipulação psicológica das vítimas para obter informações confidenciais. Isso pode incluir telefonemas falsos, phishing por telefone, pretexting e mais. Esse ataque ocorre quando um invasor insere código SQL malicioso em campos de entrada de um site para manipular o banco de dados subjacente. Isso pode resultar em vazamento de dados ou comprometimento do sistema. Um ataque de força bruta envolve tentativas repetitivas de adivinhar senhas ou chaves criptográficas até que a combinação correta seja encontrada. Isso é muitas vezes usado contra sistemas de autenticação fracos.
